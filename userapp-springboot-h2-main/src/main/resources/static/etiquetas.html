💡 ¿Qué ocurre en la BD con @ManyToMany?

Las etiquetas son el lado inverso de la relación @ManyToMany. Se crean aquí y después se asignan a las notas desde notas.html.

Hibernate genera automáticamente una tabla intermedia NOTA_ETIQUETA con dos columnas: NOTA_ID y ETIQUETA_ID. Cada fila representa un vínculo entre una nota concreta y una etiqueta concreta. No hay ninguna FK en las tablas principales.

Compruébalo en la consola H2: verás tres tablas: NOTAS, ETIQUETAS y NOTA_ETIQUETA. Ejecuta SELECT * FROM NOTA_ETIQUETA para ver cada asociación.
