💡 Relación @ManyToOne en acción
Cada nota tiene una columna USUARIO_ID que apunta a la tabla USERS. Eso es exactamente lo que configura @ManyToOne + @JoinColumn(name = "usuario_id") en Java.
Al seleccionar un usuario en el filtro, la API ejecuta:
GET /api/v1/notas/usuario/{id} → SELECT * FROM notas WHERE usuario_id = ?
Compruébalo en la consola H2: verás la tabla NOTAS con la FK USUARIO_ID.
